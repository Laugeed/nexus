# NEXUS Messenger 🚀

Твой собственный мессенджер с настоящей регистрацией, реальными чатами и передачей файлов.

## Стек
- **Бэкенд:** Node.js + Express
- **База данных:** SQLite (файл `nexus.db` — создаётся автоматически)
- **Real-time:** WebSocket
- **Файлы:** Multer (хранятся в папке `uploads/`)
- **Безопасность:** bcrypt для паролей, токены сессий

## Быстрый старт

### 1. Установи зависимости
```bash
npm install
```

### 2. Запусти сервер
```bash
npm start
```

Или с авто-перезагрузкой при изменениях:
```bash
npm run dev
```

### 3. Открой браузер
```
http://localhost:3000
```

## Как общаться с другом

**Вариант А — локальная сеть (Wi-Fi)**
1. Запусти сервер на своём компьютере
2. Узнай свой локальный IP: `ipconfig` (Windows) или `ifconfig` (Mac/Linux)
3. Друг открывает `http://ТВО_IP:3000` — должны быть в одной сети

**Вариант Б — интернет (бесплатно)**
1. Установи [ngrok](https://ngrok.com): `npm install -g ngrok`
2. Запусти: `ngrok http 3000`
3. ngrok даст тебе публичный URL типа `https://abc123.ngrok.io`
4. Отправь этот URL другу — он заходит и регистрируется

**Вариант В — деплой на сервер**
- [Railway](https://railway.app) — бесплатный хостинг для Node.js
- [Render](https://render.com) — тоже бесплатный tier

## Функционал

- ✅ Настоящая регистрация и вход (пароли хешируются bcrypt)
- ✅ Сессии с токенами (сохраняются между перезагрузкой страницы)
- ✅ Поиск пользователей по username или имени
- ✅ Добавление контактов (взаимное)
- ✅ Real-time чат через WebSocket
- ✅ Индикатор "печатает..."
- ✅ Отправка файлов любого типа (до 50 МБ)
- ✅ Скачивание файлов
- ✅ История сообщений сохраняется в БД

## API эндпоинты

| Метод | URL | Описание |
|-------|-----|----------|
| POST | /api/register | Регистрация |
| POST | /api/login | Вход |
| POST | /api/logout | Выход |
| GET | /api/me | Текущий пользователь |
| GET | /api/users/search?q= | Поиск пользователей |
| POST | /api/contacts | Добавить контакт |
| GET | /api/contacts | Список контактов |
| DELETE | /api/contacts/:id | Удалить контакт |
| POST | /api/messages | Отправить сообщение |
| GET | /api/messages/:with_id | История сообщений |
| POST | /api/files/send | Отправить файл |
